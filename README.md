FlowDay Pro

Personal Habit & Mood Intelligence PWA**

&gt; AI-powered daily journaling for working professionals. Complete your check-in under 90 seconds, discover predictive insights, and prevent burnout before it happens.

🚀 Features

- ⚡ **90-Second Check-in** — Mood, energy, habits, voice notes
- 🔮 **Predictive Insights** — AI spots patterns 7 days ahead
- 🎯 **Focus Timer** — Pomodoro with habit integration
- 🏆 **Achievement System** — Gamified streaks and milestones
- 📊 **Advanced Analytics** — Correlation analysis, sleep tracking, radar charts
- 🔒 **Privacy First** — End-to-end encryption, local-first architecture

🛠️ Tech Stack

- React + Vite
- Supabase (Auth + Database)
- Tailwind CSS
- Progressive Web App (PWA)

📦 Installation

bash
git clone https://github.com/YOUR_USERNAME/flowday-pro.git
cd flowday-pro
npm install
npm run dev
